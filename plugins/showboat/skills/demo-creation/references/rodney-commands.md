# Rodney Command Reference

Run `rodney --help` to get the full, up-to-date list of commands and options.

For help on a specific command, run `rodney <command> --help`.

## Tips

Screenshot a specific element instead of the full page:
```bash
rodney screenshot-el '.dashboard-card' demo-images/02-card.png
```

Inspect the HTML of a specific element to understand its structure:
```bash
rodney html 'form'
```
